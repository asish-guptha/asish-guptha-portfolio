# 🚀 Asish's 3D Developer Portfolio

## ✨ Key Features

* **Interactive 3D Skill Globe:** A custom-built, interactive 3D sphere rendering my exact tech stack using `Three.js` and `@react-three/fiber`. Features smooth drag-to-spin mechanics and depth-occluded floating icons.
* **Live GitHub Integration:** Dynamically fetches and displays my most recently updated repositories directly from the GitHub API.
* **In-App Resume Viewer:** A custom-built, dark-mode modal that allows recruiters to read my formatted CV seamlessly without leaving the webpage.
* **Fully Functional Contact Form:** Integrated with `EmailJS` to send messages directly to my inbox without needing a backend server.
* **Immersive Animations:** Utilizes `Framer Motion` for scroll-triggered fade-ins and a lightweight `@tsparticles` engine for a dynamic, starry background.
* **Responsive Architecture:** Perfectly optimized for mobile, tablet, and desktop viewing with customized layout stacking.

## 🛠️ Tech Stack

* **Frontend Framework:** React 18 (Bootstrapped with Vite)
* **Styling:** Tailwind CSS
* **3D Graphics:** Three.js, React Three Fiber, React Three Drei
* **Animations:** Framer Motion, tsParticles
* **Utilities:** React Scroll, React Icons
* **Form Handling:** EmailJS
* **Deployment:** Vercel

## ⚙️ Running Locally

Want to run this project on your local machine? Follow these steps:

1. **Clone the repository:**
   `git clone https://github.com/asish-guptha/asish-portfolio.git`

2. **Navigate into the directory:**
   `cd asish-portfolio`

3. **Install dependencies:**
   `npm install`

4. **Start the development server:**
   `npm run dev`

5. **Open your browser:**
   Navigate to `http://localhost:5173` to view the application.
